## 💻 How to Use the Lua Scripts

### Method 1: Using the Lua Engine (One-time execution)
1. Open **Cheat Engine**.
2. Press `Ctrl + Alt + L` to open the **Lua Engine** window.
3. Copy the code from any `lua` file in this repository and paste it into the Lua Engine.
4. Click **Execute** to run the script.

### Method 2: Using Auto-Assemble Scripts
1. Open **Cheat Engine**.
2. 2. Press `Ctrl + Alt + A` to open the **Auto Assemble** window.
3. Copy the code from any `Auto Assemble` file in this repository and paste it into the Auto Assemble.
4. 4. Click **Execute** to run the script.

### Method 3: Autorun Setup (Permanent execution)
1. Download the desired `.lua` script from this repository.
2. Move/Paste the script into your Cheat Engine **`autorun`** folder (Usually located at `C:\Program Files\Cheat Engine\autorun\`).
3. Restart Cheat Engine. The script will now load automatically every time you open Cheat Engine!
